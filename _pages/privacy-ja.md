# Pilmie プライバシーポリシー(個人情報処理方針)

施行日:2026年7月5日

Pilmie(ピルミー、以下「本サービス」)を運営するBullets(運営者:Jiyoung Ryu〔リュ・ジヨン〕、以下「運営者」)は、大韓民国の「個人情報保護法」その他の関係法令を遵守し、利用者の個人情報を保護するため、以下のとおりプライバシーポリシーを定めて公開します。

本書は理解を助けるために提供される参考訳です。韓国語で作成された文書(개인정보처리방침)が正本であり、法的拘束力を有します。韓国語版と英語・日本語等の翻訳版の解釈に相違がある場合は、韓国語版が優先します。

## 第1条(総則)

1. 本サービスは、読書記録と書籍レビューを共有するモバイルアプリケーション(iOS/Android)です。
2. 運営者は、本ポリシーに明示した目的および範囲内でのみ個人情報を処理し、目的が変更される場合には、「個人情報保護法」第18条に基づき別途の同意を取得するなど、必要な措置を講じます。

## 第2条(収集する個人情報の項目および収集方法)

運営者は、以下の個人情報を収集します。

### 1. 会員登録およびログイン時

| 区分 | 項目 | 収集方法 |
|---|---|---|
| 必須 | ソーシャルアカウントのメールアドレス、ソーシャルアカウントの氏名、会員固有識別子(UID) | Googleログイン(Android)またはAppleログイン(iOS)の連携時に自動収集 |
| 必須 | ニックネーム | 登録画面で利用者が直接入力 |
| 任意 | 一言紹介(自己紹介)、プロフィール画像 | 登録またはプロフィール編集画面で利用者が直接入力・登録 |
| 必須 | 利用規約・個人情報収集への同意の有無および同意日時、同意したポリシーのバージョン、マーケティング情報受信への同意の有無(任意同意) | 登録手続において自動記録 |

- Appleログインの利用時には、Appleが提供する「メールを非公開(Hide My Email)」機能を使用でき、その場合、実際のメールアドレスの代わりにAppleが生成した中継(リレー)メールアドレスが収集されます。
- ソーシャルアカウントのプロフィール写真は本サービスでは利用せず、電話番号、生年月日、精密（GPS）位置情報、広告識別子は収集しません。（分析目的のIPベースのおおよその地域は第3項を参照）

### 2. サービス利用の過程で利用者が生成する情報

- 書籍レビューおよび星評価、レビューへのコメント
- 書籍から引用した文章(ハイライト)およびページ情報
- 本棚情報:書籍ごとの読書ステータス(読みたい/読書中/読了)、読書進捗率、読書開始日・完了日
- フォロー/フォロワー関係、「いいね」の記録
- ブロックした利用者のリスト、通報内容(通報事由および詳細)
- アプリ内通知データ(通知の種類、関連する利用者・書籍情報等)

### 3. 自動的に収集される情報

- プッシュ通知トークン(FCMトークン):通知配信のための端末識別トークンであり、ログイン時または通知許可の取得過程で収集されることがあります。通知の許可を与えない場合、プッシュ通知は配信されません。
- エラー・障害診断情報:アプリのエラー発生時に、エラー内容、スタックトレース、端末機種・OSバージョン・アプリバージョン、エラー発生前後のアプリ利用記録(画面遷移、書籍検索リクエスト等)などの技術情報(正式リリースビルドでのみ送信され、氏名・メールアドレス等の識別情報は含まれません)
- サービス利用分析情報:画面表示および機能利用イベント(ログイン・検索・本棚への追加・レビュー作成等)とその発生時刻、アプリバージョン、端末種別、おおよその地域などの統計情報。この情報は、アプリインスタンス識別子(仮名)および会員識別子(UID)と紐づけて収集されることがありますが、サービス利用統計の作成および品質改善の目的でのみ利用し、広告識別子は収集せず、広告目的には利用しません。

### 4. 端末内にのみ保存される情報(サーバーへ送信されません)

以下の情報は利用者の端末内部にのみ保存され、運営者のサーバーへは送信されません。アプリを削除すると一緒に削除されます。

- 画面テーマ設定、最近の検索キーワード、年間読書目標、読書チェックイン(ストリーク)記録、読書リマインダー設定、オンボーディング(初回起動)完了フラグ

## 第3条(個人情報の処理目的)

| 目的 | 利用する項目 |
|---|---|
| 会員の識別、登録・ログイン処理、アカウント管理 | メールアドレス、氏名、UID、ニックネーム |
| サービスの提供(レビュー・本棚・コミュニティ機能) | ニックネーム、プロフィール画像、一言紹介、利用者が生成したコンテンツ |
| プッシュ通知およびアプリ内通知の配信(フォロー・いいね・コメント通知等) | FCMトークン、通知データ |
| マーケティング情報の配信(イベント・特典の案内)— 任意同意者に限る | FCMトークン、マーケティング受信同意の有無 |
| 不正利用の防止、通報の処理、利用者の保護 | UID、通報内容、ブロックリスト |
| サービス品質の改善およびエラー対応 | エラー・障害診断情報 |
| サービス利用統計の作成および機能改善(利用分析) | サービス利用分析情報 |
| 同意履歴の証憑 | 同意の有無・日時・ポリシーのバージョン |

## 第4条(個人情報の保有および利用期間)

1. 運営者は、利用者が退会(アカウント削除)を完了した場合、収集した個人情報を遅滞なく破棄します。
2. ただし、以下の情報は例外として、明示した期間保存します。

| 保存項目 | 保存根拠・目的 | 保存期間 |
|---|---|---|
| 通報内容(通報者・被通報者の識別子、通報事由・詳細、通報日時) | 不正利用の防止、紛争対応 | 退会後3年 |
| 関係法令により保存義務がある情報 | 当該法令 | 当該法令の定める期間 |

3. 退会時には、他の利用者のコンテンツに残っていた退会会員の痕跡(いいね、フォロー関係、レビュアー集計、退会会員が作成したレビューに付いた他の利用者のコメント等)も併せて削除処理されます。システム障害等により一部の情報が直ちに削除されなかった場合、運営者はこれを認知した時点で直ちに削除します。ただし、他の利用者のデータに含まれる退会会員に関する情報(例:他の利用者のブロックリストに保存された識別子)は当該利用者のデータとして残ることがありますが、アカウント削除後は、その識別子のみで特定の個人を識別することはできません。

## 第5条(個人情報の第三者提供)

運営者は、利用者の個人情報を第三者に提供しません。ただし、以下の場合は例外とします。

1. 利用者が事前に別途同意した場合
2. 法令の規定による場合、または捜査目的で法令に定められた手続および方法に従って捜査機関の要請があった場合

| 受託者(移転を受ける者) | 移転先の国 | 移転する項目 | 委託業務(目的) | 移転の時期・方法 | 保有期間 |

| Google LLC (Firebase) — privacy@google.com | アメリカ合衆国 | 第2条第1項〜第3項の項目全部(メールアドレス、氏名、UID、ニックネーム、プロフィール画像、コンテンツ、FCMトークン、サービス利用分析情報等) | 会員認証(Firebase Authentication)、データ保存(Cloud Firestore)、ファイル保存(Cloud Storage)、プッシュ通知配信(Firebase Cloud Messaging)、利用統計分析(Google Analytics for Firebase)、アプリ構成の提供(Firebase Remote Config) | サービス利用時点にネットワークを通じて送信 | 退会または委託契約終了時まで |

2. 利用者は、個人情報の国外移転を拒否することができます。ただし、本サービスは上記受託者の基盤インフラ上で提供されるという特性上、国外移転を拒否する場合、サービスの利用(会員登録)はできません。移転の拒否は、登録しないこと、または退会することにより行うことができ、第13条の連絡先までお問い合わせいただけます。
3. 書籍検索時には、検索キーワードがNAVER Corp.の書籍検索APIへ送信されます。その際、利用者のアカウント情報や端末識別情報は送信されず、検索キーワードのみで個人を識別することはできません。

## 第7条(個人情報の破棄手続および方法)

1. 運営者は、個人情報の保有期間の経過、処理目的の達成など、個人情報が不要となったときは、遅滞なく当該情報を破棄します。
2. 電子ファイル形式の個人情報は、復元・再生が不可能な方法で完全に削除します。
3. 退会は、アプリ内の[設定 → アカウント削除]メニューから直接行うことができ(セキュリティのため、最終ログインから一定時間が経過している場合、本人確認のための再ログインが求められることがあります)、退会時には、プロフィール、レビュー、星評価、コメント、引用文章、本棚記録、フォロー関係、いいね、ブロックリスト、通知、プロフィール画像ファイルが削除され、認証アカウントが削除されます(第4条第2項の保存例外を除く)。

## 第8条(情報主体の権利・義務およびその行使方法)

1. 利用者は、いつでも自己の個人情報について、閲覧、訂正、削除、処理停止を求めることができます。
2. 権利の行使方法:
   - プロフィール情報(ニックネーム、一言紹介、プロフィール画像)の閲覧・訂正:アプリ内のプロフィール編集機能
   - コンテンツ(レビュー・コメント・文章等)の修正・削除:アプリ内の当該コンテンツの修正・削除機能
   - アカウントおよび全データの削除:アプリ内の[設定 → アカウント削除]
   - その他の請求(閲覧・処理停止等):第13条の個人情報保護責任者へメールで請求
3. メールで受け付けた請求について、運営者は遅滞なく(遅くとも10日以内に)措置し、その結果を通知します。
4. 権利の行使は、利用者の法定代理人または委任を受けた者を通じて行うこともでき、この場合、適法な委任関係を確認できる書面を提出しなければなりません。
5. 利用者は、自己の個人情報を最新の状態に保つ責任を負い、他人の個人情報を盗用して登録した場合には、会員資格の喪失および関係法令に基づく処罰を受けることがあります。

## 第9条(満14歳未満の児童の個人情報)

本サービスは満14歳以上の利用者のみ登録することができ、満14歳未満の児童の個人情報は収集しません。登録手続において満14歳以上であることの確認を受け、満14歳未満の児童が登録した事実が確認された場合には、当該アカウントおよび個人情報を遅滞なく削除します。

## 第10条(自動収集装置の設置・運用およびその拒否)

1. 本サービスはモバイルアプリとして提供され、クッキー(cookie)は使用しません。
2. プッシュ通知は、利用者が通知の許可を与えた場合にのみ配信され、利用者は端末の通知設定またはアプリ内設定から、いつでも通知の受信を拒否することができます。通知を拒否しても、通知の受信以外のサービス利用に制限はありません。
3. エラー診断情報の収集はサービス品質の維持のためのものであり、個人を識別できる情報は含まれません。
4. 広告識別子(ADID/IDFA)は収集せず、広告配信のための広告SDKは使用しません。
5. 本サービスは、利用統計の作成および品質改善のためにGoogle Analytics for Firebaseを使用します。このツールは、画面表示・機能利用イベント等の利用情報を仮名(アプリインスタンス)識別子に基づいて収集し、これは会員識別子(UID)と紐づけられることがあります。広告識別子は収集せず、広告目的には利用せず、収集された情報はサービス利用統計および品質改善の目的でのみ利用されます。利用者は、第13条の連絡先を通じて、利用分析情報の処理停止を請求することができます。

## 第11条(コンテンツの公開に関する案内)

1. 本サービスは、読書記録を共有するコミュニティサービスです。利用者のプロフィール(ニックネーム、一言紹介、プロフィール画像、フォロワー・フォロー中リスト)および利用者が作成したコンテンツ(レビュー、星評価、コメント、引用文章、本棚の読書ステータス)は、本サービスの他の利用者に公開されます。
2. 公開を望まない情報はコンテンツに含めないようご注意ください。利用者は、いつでも自己のコンテンツを修正・削除することができます。
3. ブロックリストおよび通報内容は、他の利用者には公開されません。

## 第12条(個人情報の安全性確保措置)

運営者は、個人情報の安全性を確保するため、以下の措置を講じています。

1. 通信区間の暗号化:すべてのデータはHTTPS(TLS)暗号化通信で送信されます。
2. アクセス制御:データベースのセキュリティルール(Firebase Security Rules)により、本人データへの書き込み権限を本人に制限し、ブロックリスト等の機微なデータは本人のみが閲覧できるよう制御します。
3. アクセス権限の最小化:個人情報にアクセスできる管理権限を運営者に最小化します。
4. 収集の最小化:サービス提供に必要な最小限の情報のみを収集します。

## 第15条(プライバシーポリシーの変更)

1. 本ポリシーの内容が追加、削除または修正される場合、施行の7日前(利用者に不利な変更または重大な変更の場合は30日前)から、アプリ内のお知らせまたはポリシー掲載ページを通じて告知します。
2. 本ポリシーの以前のバージョンは、請求により第13条の連絡先を通じて確認することができます。
